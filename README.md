# Jogo da Velha 

O **Jogo da Velha** é um jogo de tabuleiro para dois jogadores, onde um jogador usa o símbolo X e o outro O. 

O objetivo é formar uma linha com três símbolos iguais na horizontal, vertical ou diagonal, enquanto impede o adversário de fazer o mesmo. O jogo termina quando um jogador vence ou quando todas as casas do tabuleiro são preenchidas, resultando em empate se ninguém ganhar.
